Competency theme - Installation steps

  • On your downloaded package, you will find theme_competency.zip
  • Unzip - theme_competency.zip, you will get folder 'competency'
  • Copy folder 'competency' and put into theme folder of your moodle system
  • Next login as Site administrator
  • Go to 'Site administration' -> 'Notifications' , here on 'Plugins check' page you will see the competency theme in listing.
  • Click the "Upgrade Moodle database now" button displayed on bottom of the page
  • You will get a success message once the theme is installed successfully.
  • By clicking "Continue" button on success page , you will get options to change the competency theme settings.
  • You can modify the settings later too :-)

Steps to set Competency theme as default theme for your Moodle system

  • Login as site administrator
  • Go to Site administration -> Appearance -> Themes -> Theme selector
  • Click the "Change theme" button for device type as "Default"
  • It will list all the available themes for "Default device"
  • Then click "Use theme" button on "Competency theme"
  • Next click the "Continue" button, thats it!
  • Cheers,you have done it !!!
  • If you need any support related to this theme , kindly send an email to info@debonairtraining.com

Steps to configure Competency theme

ul>
  • Login as site administrator
  • Go to Site administration -> Appearance -> Competency Themes
  • Click the "General Setting" And set all necessary settings
  • Click the "Custom Image Setting" to set the favicon image, Header image and other images
  • Click the "Icon Navigation" to set the icons menu
  • Click the "Marketing Tiles" to set the promoted courses or courses you may be interested in
  • Click the "Categories Tiles" to set the categories
  • Follow the steps above to carry out other settings
  • Cheers, you have done it !!!

    • Developer Documentation

    • Front page
    • The header image is dynamically uploaded from the theme settings: Site administration --> Themes --> Competency theme --> Then Custom image tab for Header Image.
    • The Colour scheme can be dynamically set using the colour pickers in: Site administration --> Themes --> Competency theme. Note SCSS can be added, or directly in the scss files for advanced developers.
    • The header Title "Get Competitive Advantage with E-Learning" can be changed to one of choice in others.mustache
    • The best, worst and competency path are programmed in the core_renderer.php, dashboard_matrix function and dashboard.mustache files
    • The Search bar is in the core_renderer and search.mustache.
    • The Metrics, Time, Certification and classes icons can be changed in the core_renderer and fpicons.mustache
    • Categories and Marketing tiles can be dynamically uploaded from the theme settings site administration --> Themes --> Competency theme --> "Marketing Tiles" "Categories Tiles" tabs respectively.
    • Default front page settings still apply after any of the above enabled functionalities are enabled / rendered.
    • The Footer is setup in footer.mustache

    • Dashboard
    • The dashboard comes with default theme_boost blocks of Timeline and Calendar. Blocks can be added by clicking on the blocks drawer. They can be removed at Site Administration -> Plugins -> Blocks and hiding the unwanted blocks, for example Calendar
    • My Courses
    • All the courses the user is enrolled in; will show here by default and it is similar to the course overview block which can be added to the dashboard
    • Theme settings
    • Theme settings including a Custom Menu can be added/adjusted similar the method of adding it in Moodle3.x
